# Design System Audit

Audit the current UI implementation against the design system.

Check all components for:
- Correct color token usage (no hardcoded colors)
- Typography consistency (correct fonts, sizes)
- Spacing adherence to the scale
- Animation curves matching the system
- Responsive behavior at 1024px, 768px breakpoints
- Hover/focus/active states on all interactive elements
- Empty, loading, and error states

Use the qa-reviewer agent for a thorough review.
Report findings categorized as Must Fix / Should Fix / Nice to Have.
